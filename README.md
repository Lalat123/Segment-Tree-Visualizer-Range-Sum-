# 🌳 Segment Tree Visualizer (Range Sum)

An interactive web-based tool to visualize the construction and operations of a **Segment Tree** for Range Sum Queries. Built with vanilla JavaScript and SVG.

## 🚀 Live Demo
[Click here to use the Visualizer](https://segment-tree-visualizer-1--lalat-kesharike.replit.app/)

## ✨ Key Features
* **Dynamic Construction:** Input any array (e.g., `1 2 3 4 5`) to instantly generate the tree structure.
* **Visual Feedback:** Uses SVG to render nodes and edges, creating a clear hierarchical view of the recursion.
* **Range Sum Query:** Calculate the sum of elements in index range `[L, R]` and verify the logic.
* **Point Updates:** Update a value at a specific index and visualize how the change propagates up the tree.

## 🛠️ Tech Stack
* **Core:** HTML5, CSS3
* **Logic:** Vanilla JavaScript (No external libraries)
* **Graphics:** Scalable Vector Graphics (SVG) for rendering the tree.
